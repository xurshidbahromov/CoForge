# Wireframes (Low‑Fidelity Sketches)

## 1. Onboarding Survey
- Full‑screen modal with glassy background.
- Three sections: Technology Stack (checkboxes), Experience Level (radio), Goal (dropdown).
- "Next" button with subtle hover glow.

## 2. Project Hub
- Dashboard layout with a responsive grid of project cards.
- Each card uses a semi‑transparent glass effect, rounded corners, and a subtle drop‑shadow.
- Hover reveals "View" and "Join" buttons.

## 3. Solo Flow – Idea Generator
- Centered card showing AI‑generated project title and description.
- "Accept" and "Regenerate" buttons.

## 4. Task Breakdown & Progress Tracker
- Vertical list of tasks with checkboxes.
- Progress bar at the top (glass gradient).

## 5. Code Review Feedback
- Split view: code editor on left, AI feedback panel on right with glassy overlay.
- Inline suggestions highlighted.

## 6. Team Flow – Kanban Board
- Three columns (To Do, In Progress, Done) with draggable cards.
- Card design matches project cards (glass, subtle animation).

## 7. Profile / Proof of Experience
- Profile page with sections for Solo Projects and Team Projects.
- Each project displayed as a glass card with tags for stack, role, and AI summary.

**Design Notes**
- Use a harmonious pastel‑blue / teal palette with dark mode variations.
- Typography: "Inter" for body, "Playfair Display" for headings.
- Micro‑animations: fade‑in, scale on hover, smooth transitions.
